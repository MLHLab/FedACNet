FedACNet: A federated attention-driven convolutional neural network-based framework for the detection of pneumonia-induced lung infections over distributed data

Various chest X-ray images across different countries (specific to three different continents) were acquired to form a curated repository. Following this, data augmentation was implemented to overcome the class-imbalance effect on the model performance. A Federated Attention-driven Convolutional neural Network (FedACNet) framework was developed for classification of pneumonia-triggered lung infections over such distributed data, following the centralized FL topology with federated averaging (FedAvg) optimization. The objective of our present work is to investigate the following aspects: (A) Exploring the utilization of a FL architecture for constructing an AI-driven solution to detect pneumonia on a global scale. (B) Comparing the performance of the proposed classification model and FL framework with existing methodologies to establish its efficacy. (C) Referencing a cloud-based architecture that can support the development of such models on a global scale.
